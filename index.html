Min Resa

    Min resa som utvecklare började med att jag kom i kontakt med TI-BASIC språket som fanns på miniräknarna när jag läste Matematik. Fascinationen för att automatisera uträkningar var det som startade allt.

    Jag började programmera i Java vid slutet av 2016. Fokuset i början hamnade på att skapa botar av olika slag. Därefter det spred sig att bli mer omfattande och blandannat inkludera Grafiska Användargränsnitt, WebbAutomatisering, och Backend utveckling med Ramverket Spring.

    Ett intresse för Computer Vision och Machine Learning gjorde att jag började programmera i Python. Här skapade jag program baserat på Computer vision för att följa kulor i Roulette, och utforskade även Facial Recognition.

    Efter att ha skapat ett program för att buda hem saker på Tradera bestämde jag mig för att börja kartlägga information om inköpen. Valet hamnade på MySQL via JDBC.

  • 2019: Django & SQLite

    Django och SQLite var ett utav de första ramverken jag använde mig för att skapa WebbApplikationer.
